# 💰 급여관리 자동화

**2026년 기준 4대보험 및 소득세 자동 계산 시스템**

## 📌 주요 기능

### 1. 💵 급여 계산
- **4대보험 자동 계산**
  - 국민연금 (4.5%)
  - 건강보험 (7.095%) + 장기요양보험 (12.82%)
  - 고용보험 (1.8%)
  - 산재보험 (업종별)

- **소득세 자동 계산**
  - 근로소득세 (간이세액 기준)
  - 지방소득세 (소득세의 10%)
  - 부양가족 수 반영

- **실수령액 계산**
  - 총 지급액 - 공제액 = 실수령액

### 2. 📊 급여대장 관리
- 월별 급여 요약
- 부서별/직급별 통계
- 엑셀 다운로드

### 3. ⏰ 시간외 수당
- 연장근로 (50% 가산)
- 야간근로 (50% 가산)
- 휴일근로 (50~100% 가산)
- 자동 계산 및 관리

### 4. 📅 연차 관리
- 입사일 기준 연차 자동 계산
- 1년 미만: 월 1개
- 1년 이상: 년 15개
- 3년 이상: 매 2년마다 1개 추가 (최대 25개)
- 미사용 연차수당 계산

### 5. 💼 퇴직금 계산
- 평균임금 자동 계산 (퇴직 전 3개월)
- 퇴직금 = (평균임금 × 30일 × 재직일수) / 365
- 근속 연수 자동 계산

```bash
cd 5_급여관리_자동화

```

## 📋 사용 순서

1. **급여 설정** (⚙️ 메뉴)
   - 직원별 기본급 입력
   - 수당 설정 (식대, 교통비 등)
   - 비과세 항목 선택
   - 부양가족 수 입력

2. **월별 급여 계산** (💰 메뉴)
   - 직원 선택
   - 자동 계산 확인
   - 급여 이력에 저장

3. **급여대장 확인** (📊 메뉴)
   - 월별 전체 직원 급여 조회
   - 엑셀 다운로드

4. **시간외 수당** (⏰ 메뉴)
   - 연장/야간/휴일 근무 등록
   - 자동 수당 계산
