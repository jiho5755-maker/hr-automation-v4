## 📦 포함된 시스템

### 🏢 통합 대시보드
- 📋 직원 관리
- 🏢 회사 정보 관리
- 🤰 출산·육아 날짜 관리

### 🔗 연결된 자동화 앱
1. **🤰 출산육아 자동화**
   - 임신/출산/육아휴직 서류 자동 생성
   - 근무일지 자동 작성
   - 정부 지원금 계산

2. **💰 연말정산 자동화**
   - 국세청 PDF 자동 추출
   - 엑셀 매핑
   - 주택 정보 파싱

3. **🏠 재택근무 관리시스템**
   - 근무 로그 관리
   - 캘린더 뷰
   - 통계 대시보드

4. **🏛️ 정부지원금 자동화**
   - 맞춤형 지원금 검색
   - 신청 이력 관리
   - 스마트 가이드 생성

5. **💵 급여관리 자동화**
   - 4대보험 자동 계산
   - 소득세 자동 계산
   - 급여명세서 생성
   - 급여대장 생성

## 🔧 문제 해결

### ❌ "command not found: streamlit"

**해결**:
```bash
pip3 install streamlit
```

또는 `🚀_인사팀_자동화_시작.command` 파일을 더블클릭하세요.

---

### ❌ 포트 8000이 이미 사용 중

**해결**:
```bash
# 다른 포트 사용
python3 -m streamlit run app.py --server.port 8001
```

---

### ❌ 페이지가 로드되지 않음

**해결**:
1. 터미널에서 **Ctrl + C**로 종료
2. `🚀_인사팀_자동화_시작.command` 파일을 다시 더블클릭
3. 브라우저 캐시 삭제 후 새로고침

## 💡 팁

### 개발 모드 실행
```bash
streamlit run app.py --server.runOnSave true
```

### 외부 접속 허용
```bash
streamlit run app.py --server.address 0.0.0.0
```

### 테마 변경
```bash
streamlit run app.py --theme.base light
```
