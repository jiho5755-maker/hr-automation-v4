## 🚀 실행 방법

### **1. 통합 대시보드 (가장 쉬움)** ⭐

```bash
cd ~/Documents/인사팀_자동화_마스터/0_통합_대시보드
./🚀_실행하기.command
```

**접속**: http://localhost:8000  
**로그인**: `admin` / `admin1234`

### **2. 개별 앱 실행 (기존 방식)**

```bash
# 출산육아 (8501)
cd 1_출산육아_자동화
./🚀_실행하기.command

# 연말정산 (8502)
cd 2_연말정산_자동화
./🚀_실행하기.command

# 재택근무 (8503)
cd 3_재택근무_관리시스템
./🚀_실행하기.command

# 정부지원금 (8504)
cd 4_정부지원금_자동화
python3 -m streamlit run app_v3.py --server.port 8504
```

### **3. Docker로 실행 (Windows 홈서버)**

```bash
# 전체 시스템 시작
docker-compose up -d

# 로그 확인
docker-compose logs -f

# 중지
docker-compose down
```
