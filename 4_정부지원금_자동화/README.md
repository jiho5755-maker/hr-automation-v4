# 🏢 회사 맞춤형 정부 지원금 자동화

우리 회사 상황에 딱 맞는 정부 지원금을 자동으로 찾아주는 시스템입니다.

## ✨ 주요 기능

### 🔍 지원금 자동 매칭
- 회사 프로필 기반 지원금 검색
- 매칭도 점수 자동 계산 (0~100%)
- 예상 수령액 자동 계산

### 📊 상세 정보 제공
- 신청 조건 자동 안내
- 필요 서류 목록 표시
- 신청 기한 알림
- 문의처 정보 제공

### 💼 회사 프로필 관리
- 회사 정보 저장/재사용
- 우선지원대상기업 자동 판단
- 직원 현황 관리

## 💰 지원 지원금 (예시)

### 출산·육아 관련
- **출산전후휴가 대체인력 지원금**: 월 140만원 × 15개월 = 2,100만원
- **임신기 근로시간 단축 지원금**: 월 40만원 × 12개월 = 480만원
- **일자리 함께하기 지원금**: 월 60만원 × 6개월 = 360만원

### 청년고용 관련
- **청년 일자리 도약장려금**: 월 80만원 × 12개월 = 960만원
- **청년 내일채움공제**: 2년간 적립금 지원

**총 예상 수령액: 약 3,900만원+**

## 🚀 사용 방법

1. **회사 프로필 탭**: 회사 정보 입력 및 저장
2. **지원금 검색 탭**: 검색 버튼 클릭
3. **결과 확인**: 매칭된 지원금 목록 및 상세 정보

## 🛠️ 기술 스택

- **Frontend**: Gradio 4.44
- **Backend**: Python 3.9
- **Data**: Pandas, JSON
