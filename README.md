## 💡 사용 시나리오

### 시나리오 1: 임신 직원 지원금 신청
```
1. 출산육아 자동화 실행
2. 직원 정보 입력
3. "임신기 근로시간 단축 신청서" 선택
4. PDF 다운로드 → 정부 기관 제출
5. 예상 수령액: 480만원
```

### 시나리오 2: 회사 맞춤 지원금 검색
```
1. 정부지원금 자동화 실행
2. 회사 프로필 입력 (최초 1회)
3. "지원금 검색" 버튼 클릭
4. 매칭된 지원금 목록 확인
5. 예상 수령액: 3,900만원+
```

### 시나리오 3: 연말정산 데이터 추출
```
1. 연말정산 자동화 실행
2. 국세청 PDF 업로드
3. 자동 파싱 완료
4. 엑셀 다운로드 → 회계팀 전달
5. 소요 시간: 5분
```

### 시나리오 4: 재택근무 일정 관리
```
1. 재택근무 관리시스템 실행
2. 로그인 (관리자 또는 직원)
3. 일정 등록 또는 출퇴근 기록
4. 월간 리포트 다운로드
5. 소요 시간: 6분
```
