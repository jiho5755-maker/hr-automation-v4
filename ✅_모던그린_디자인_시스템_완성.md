## 📦 새로운 공통 모듈

### 1️⃣ `shared/design.py`
**통일된 디자인 시스템 제공**

```python
from shared.design import apply_design

# 모든 앱에서 호출하면 자동으로 디자인 적용
apply_design()
```

#### 주요 기능:
- **컬러 시스템**: 
  - Primary: 모던 그린 (#059669, #10b981)
  - Status: Success, Warning, Error, Info
  - Neutral: 미니멀한 회색 계열
  
- **타이포그래피**:
  - Inter, Pretendard 폰트 기반
  - 명확한 계층 구조
  
- **컴포넌트 스타일**:
  - 깔끔한 카드 디자인
  - 소프트 쉐도우 (미니멀)
  - 부드러운 hover 애니메이션
  - 그라데이션 버튼
  - 모던한 input/select 박스

### 2️⃣ `shared/utils.py`
**성공/실패 메시지 토스트 알림**

```python
from shared.utils import show_success, show_info, show_warning, show_error

# 기존: st.balloons() ❌ (너무 짜증나는 이펙트)
# 신규: show_success() ✅ (심플하고 모던한 토스트)

show_success("직원이 추가되었습니다!")
show_info("정보가 업데이트되었습니다.")
show_warning("입력값을 확인해주세요.")
show_error("저장에 실패했습니다.")
```

#### 특징:
- 🍞 **Toast 알림**: 화면 우측 하단에 부드럽게 나타났다 사라짐
- 🎯 **심플함**: 풍선 이펙트 대신 깔끔한 메시지만 표시
- ⚡ **빠른 피드백**: 사용자 행동에 즉각 반응
- 🎨 **일관성**: 모든 앱에서 동일한 스타일

## 🌿 디자인 시스템 상세

### 컬러 팔레트

```css
/* Primary - 모던 그린 */
#059669  /* 메인 */
#10b981  /* 강조 */
#047857  /* 호버 */

/* Status */
#10b981  /* Success - 초록 */
#f59e0b  /* Warning - 오렌지 */
#ef4444  /* Error - 빨강 */
#3b82f6  /* Info - 파랑 */

/* Neutral - 미니멀 */
#f8fafc  /* 배경 */
#ffffff  /* 카드 */
#e2e8f0  /* 테두리 */
#1e293b  /* 진한 텍스트 */
#64748b  /* 보조 텍스트 */
```

### 타이포그래피

```css
/* 폰트 */
font-family: 'Inter', 'Pretendard', -apple-system, BlinkMacSystemFont, system-ui, sans-serif

/* 크기 */
H1: 32px (xxlarge) - 그라데이션 적용
H2: 24px (xlarge) - 모던 그린
H3: 18px (large)
Base: 16px
Small: 14px
```

### 컴포넌트

#### 버튼
- 그라데이션 배경 (Primary → Primary Light)
- 12px 둥근 모서리
- 부드러운 hover 효과 (transform + shadow)
- 소프트 쉐도우

#### 카드
- 흰색 배경
- 12px 둥근 모서리
- 1px 연한 테두리
- 소프트 쉐도우
- hover 시 살짝 위로 이동

#### Input/Select
- 2px 테두리 (연한 회색)
- focus 시 모던 그린 테두리
- 부드러운 focus ring

#### Metric 카드
- 큰 숫자는 모던 그린
- 라벨은 보조 텍스트 색상
- 충분한 여백

## 🚀 사용 방법

### 새로운 앱에 적용하기

```python
import streamlit as st
import sys
from pathlib import Path

# shared 모듈 import
sys.path.append(str(Path(__file__).parent.parent))
from shared.design import apply_design
from shared.utils import show_success, show_error

# 페이지 설정
st.set_page_config(
    page_title="새로운 앱",
    page_icon="🎉",
    layout="wide"
)

# 디자인 적용 (이것만 호출하면 끝!)
apply_design()

# 앱 코드...
st.title("🎉 새로운 앱")

if st.button("저장"):
    # 저장 로직...
    show_success("저장되었습니다!")  # 토스트 알림!
```

### 커스텀 스타일 추가하기

```python
# 기본 디자인 적용 후
apply_design()

# 추가 커스텀 스타일
st.markdown("""
<style>
    .my-custom-class {
        /* 여기에 추가 스타일 */
    }
</style>
""", unsafe_allow_html=True)
```
