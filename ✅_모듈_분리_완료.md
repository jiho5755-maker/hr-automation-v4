## 🔄 통합 DB 공유

**중요**: 모든 모듈이 `hr_master.db`를 공유합니다!

### 직원 정보 흐름
```
1. 직원 관리 (8000)에서 직원 추가
   ↓
2. hr_master.db에 저장
   ↓
3. 모든 모듈에서 자동으로 사용 가능!
   - 출산육아 (8501)
   - 재택근무 (8503)
   - 급여관리 (8505)
```

### 장점
- ✅ **한 번만 입력**: 직원 정보 중복 입력 불필요
- ✅ **자동 동기화**: 실시간 반영
- ✅ **데이터 일관성**: 모든 모듈에서 동일한 정보
