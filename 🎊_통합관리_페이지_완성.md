#### 직원 정보 수정:
1. "직원 목록" 탭 선택
2. 수정할 직원 선택
3. 정보 수정
4. "💾 저장" 버튼 클릭

→ **모든 앱에 자동 반영!** ✨

### **3. 회사 정보 관리**

**왼쪽 사이드바 → 🏢 회사 정보 관리 클릭**

#### 회사 정보 등록/수정:
1. "회사 정보 수정" 탭 선택
2. 회사 정보 입력
   - 회사명, 대표자명, 사업자번호 (필수)
   - 직원 수, 연매출
   - 주소, 전화번호
   - 회사 상황 (정부지원금용)
3. "💾 저장" 버튼 클릭

→ **모든 앱에 자동 반영!** 🎊
